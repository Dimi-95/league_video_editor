A program which downloads a League of Legends VOD from Twitch, goes through it, detects "highlights" based on kills, deaths and assists, and edits everything together into a montage video.

Dependencies:
-
- OpenCV
- pytesseract
- os
- ctypes
- turtle
- pydoc
- glob
- shutil
- tkinter
- time
